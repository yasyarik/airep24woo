AiRep24 connects WooCommerce stores to the AiRep24 assistant platform.

== Description ==

AiRep24 for WooCommerce installs the storefront assistant widget, syncs store knowledge, exposes assistant and widget settings in WordPress admin, and connects billing/trial through AiRep24.

== Features ==

* Connect WooCommerce store to AiRep24.
* Insert assistant widget automatically.
* Configure avatar, widget colors, position, voice mode and welcome message.
* Sync products, pages and theme color signals.
* Send product update events to AiRep24.
* Open AiRep24 trial and billing checkout from WordPress admin.

== Installation ==

1. Upload the plugin ZIP or install it from WordPress admin.
2. Activate the plugin.
3. Open AiRep24 in WordPress admin.
4. Connect AiRep24 or start a trial.
5. Sync store knowledge and configure the widget.
